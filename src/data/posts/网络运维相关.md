# ajax 之前

ajax 之前，服务端返回整个 HTML
ajax 不是新的编程语言，而是一种使用现有标准的新方法
ajax 最大的优点是在不重新加载整个页面的情况下，可以与服务器交换数据

# 互联网

电脑 - 路由器 - 调制解调器（modem）- 电话线 - 路由器 - 电脑

# 浏览器到服务器

输入域名
DNS 解析域名并返回正确 ip
本机 ip 与服务器 ip 成功建立 TCP/IP 通讯
本机浏览器接收到 html js css 并开始渲染
